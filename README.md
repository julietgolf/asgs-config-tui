# Install
## Linux
```bash
git clone git@github.com:julietgolf/asgs-config-tui.git
python -m venv .venv
. .venv/bin/activate
pip install uv && uv pip install .
```

## Windows

```powershell
git clone git@github.com:julietgolf/asgs-config-tui.git
python -m venv .venv
.venv/Scripts/activate
pip install uv
uv pip install .
```

# Usage
```
asgs-config --help
asgs-config
```
